¿Como funciona?

Al momento de invocar la clase grid internamente se crea un contenedor en el cual se espera x número de contenedores con clase "grid-row" (Filas),luego dentro de estas se espera x número de elementos con clase "col-lg-x" (siendo x el número de columnas). En el grid de Guava existe un máximo de 12 columnas las cuales equivalen a un porcentaje del ancho total del contenedor padre; ejemplo: col-lg-3 equivale a asignarle un ancho del 25% del contenedor padre todo esto siguiéndose de una regla de 3 simple;
col-lg-12 -> 100%
col-lg-x -> n
n=(x * 100)/12

Margin/Padding
El margin se maneja con mrg- y el padding con pd- concatenandole el valor de dicha pripiedad q va desde 0 hasta 200 de 5 en 5 adicional a esto, si se requiere añadirle una direccion como por ejemplo left o right seria de la siguiente manera ;
mrg-l-: margin left
mrg-r-: margin right
mrg-t-: margin top
mrg-b-: margin bottom
mrg-a-: margin auto
y lo mismo seria para el padding
pd-l-: padding left
pd-r-: padding right
pd-t-: padding top
pd-b-: padding bottom
pd-a-: padding auto

Grid
El grid usa la clase grid,esta clase se usa para crear tablas o posicionar elemntos en froma de cuadricula;
Grid-row
Para crear filas en un grid solo hay q usar usa la clase grid-row
Col
Para crear columnas en un grid solo hay q usar usa la clase col- esta clase debe ser hija de grid-row para q funciona coreptamente esta clase tiene varias propiedades,pero antes de eso se debe aclarr q solo se puede crear una maximo de 12 columnas siendo este el 100% de ancho de su padre;
Ls columnas se dividen en dos partes:
col- lg- 3: El lg- se refiere al tipo de dipositivo,siendo este para dispositivos grandes como pc, el valor numerico indica cuantas columnas va a ocupar ese elemento.
Los valores para adaptarlo a los dispocitivos son :
lg Grande
md Mediano (Prtatiles)
sm Pequeño (tablets)
xs Extra Pequeño (telefonos)
space
Para crear un spaciado antes o despues de una columna se usa space-, este al igual q las columnas necesitan expesificarle en q dispositivo se van a usar
lg Grande
md Mediano (Prtatiles)
sm Pequeño (tablets)
xs Extra Pequeño (telefonos)
Luego la posicion antes b-, despues a- o q no tenga expacio n y finalmente el numero de columnas;ejemplo space-lg-b-2

Esta propiedad fuerza a q se respete la forma de tabla,simplemente se usa clear,esto es en caso dado de q los elelmntos de una fila salten a otra
